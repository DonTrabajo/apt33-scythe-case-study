# Attack Flow Builder (Public-Safe)

## Objective
Document how to transform threat research into a structured attack flow without exposing operational details.

## Tool
Attack Flow Builder (Center for Threat-Informed Defense)
Website: attackflow.org

## Process Summary
- Create actions with behavior-focused names (e.g., Discovery, Persistence, Exfiltration).
- Map actions to MITRE ATT&CK tactics and techniques.
- Add short descriptions and evidence fields for each action.
- Export a diagram file for review and collaboration.

## Notes
This summary intentionally omits commands, credentials, and target-specific data.
